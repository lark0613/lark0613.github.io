# AI 核心概念与应用入门笔记

## 一、核心概念速览
- **人工智能（AI）**：模拟人类智能的技术，核心是让机器具备感知、思考、决策和学习能力。
- **机器学习（ML）**：AI 的核心分支，通过数据训练模型，让机器自主学习规律而非硬编码规则。
- **深度学习（DL）**：机器学习的子集，基于神经网络（多层感知器），擅长处理图像、语音等复杂数据。
- **大语言模型（LLM）**：深度学习的应用，专注自然语言处理（NLP），如 GPT、LLaMA 系列，能理解和生成人类语言。
- **prompt 工程**：通过优化输入指令（提示词），让 AI 更精准输出结果的技巧，是高效使用 AI 的关键。
- **多模态 AI**：融合文本、图像、语音、视频等多种数据类型的模型，如 DALL·E、Gemini，支持跨模态交互。

## 二、常用 AI 工具分类
### 1. 通用大模型
- ChatGPT（OpenAI）：擅长对话、写作、代码生成、逻辑推理，支持多轮交互。
- 文心一言（百度）：适配中文场景，支持文本、图像、语音等多模态交互，侧重国内合规应用。
- 通义千问（阿里）：侧重企业级应用与多任务处理，支持 API 集成到各类系统。
- Claude（Anthropic）：以安全性和长文本处理见长，支持上传文档进行分析。

### 2. 专项工具
- **代码生成**：GitHub Copilot（辅助编程，支持主流语言）、CodeLlama（Meta 开源，专注代码场景）。
- **图像生成**：Midjourney（艺术创作，需 Discord 使用）、Stable Diffusion（开源可本地部署）、DALL·E 3（OpenAI，文本转图像精度高）。
- **办公效率**：Notion AI（文档编辑、总结、翻译）、飞书妙记（语音转文字+智能总结）、WPS AI（表格分析、PPT 生成）。
- **视频生成**：Runway ML（文本转视频、视频编辑）、Pika Labs（高清视频生成，支持风格定制）。

## 三、实用 prompt 编写技巧
1. **明确需求边界**：限定输出格式、场景和长度。例：“写一篇 300 字内的职场周报提纲，分工作成果、待办事项、问题反馈三部分，语言简洁正式”。
2. **补充背景信息**：提供上下文让 AI 精准适配。例：“我是前端开发，用 React 框架，帮我解释 hooks 中 useState 和 useReducer 的区别，结合 5 行内简单代码示例”。
3. **指定输出风格**：适配使用场景。例：“以 10 岁儿童能理解的语气，科普什么是 AI 绘画，不用专业术语，用比喻说明”。
4. **分步引导思考**：复杂问题拆分成多步骤指令。例：“先分析用户需求‘做一款老年人用的购物 App’的核心痛点，再列出 3 个产品功能设计建议”。

## 四、学习资源推荐
### 1. 入门书籍
- 《人工智能：现代方法》：AI 基础理论经典教材，覆盖逻辑、概率、学习等核心模块。
- 《深度学习入门》：Python 实战导向，适合零基础入门深度学习。
- 《大语言模型应用开发实战》：聚焦 LLM 应用落地，含 API 调用、prompt 工程等实操内容。

### 2. 在线课程
- Coursera《Machine Learning》（Andrew Ng 主讲）：机器学习入门标杆课程，数学推导清晰。
- B 站“李沐动手学深度学习”：开源免费，侧重代码实战，适合工程导向学习者。
- 极客时间《大模型应用开发实战》：讲解 LLM 微调、RAG 等进阶技术，适配国内场景。

### 3. 开源项目与工具
- Hugging Face：全球最大的 AI 模型库，提供预训练模型、数据集和开发工具。
- LangChain：LLM 应用开发框架，简化对话机器人、知识库问答等场景搭建。
- Llama.cpp：轻量级 LLM 部署工具，支持在普通电脑上运行开源大模型。

### 4. 资讯平台
- 机器之心、AI 前线：国内 AI 领域权威资讯，覆盖技术动态和产业应用。
- GitHub Trending（AI 分类）：追踪全球最新热门 AI 开源项目。
- arXiv（cs.AI 分类）：查看 AI 领域最新研究论文，了解前沿技术方向。

## 五、注意事项
1. **数据安全**：不向 AI 工具泄露密码、隐私数据、商业机密等敏感信息，避免数据泄露风险。
2. **结果校验**：AI 输出可能存在“幻觉”（虚假信息）、代码 bug 等问题，关键场景（如学术写作、代码上线）需人工核实。
3. **伦理边界**：避免使用 AI 生成侵权内容、虚假信息、恶意代码或违反公序良俗的内容。
4. **版权问题**：使用 AI 生成的图像、文本等内容时，确认工具版权协议，商用场景需获得授权。
